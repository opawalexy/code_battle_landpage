Code Battle: A 7-Year-Old's Coding Dream

Empowering Kids to Learn Coding, One Card at a Time

A card game designed by Timi, a 7-year-old coder, to help kids and families learn coding without a computer!

The Inspiring Story of Timi

During the Covid-19 pandemic lockdown, Timi was just 4 years old. While his dad worked on coding projects, Timi would sit quietly beside him, watching intently. One day, to his dad's surprise, Timi started writing code on a whiteboard. Even without formal teaching, he had learned by observing!

Timi's dad decided to teach him coding, and together they explored creative ways to learn. Timi came up with the idea of Code Battle, a fun and interactive way for kids and adults to learn coding concepts. Now, at 7 years old, Timi dreams of empowering kids worldwide—especially in places without access to computers or electricity—to learn coding.

This is how Code Battle was born: A card game that brings coding to life for everyone!

Why Choose Code Battle?

  • Teaches foundational coding skills like HTML structure and syntax.
  • Fun and engaging for kids (ages 4+) and families.
  • Screen-free learning experience for classrooms and at home.
  • Builds problem-solving, critical thinking, and digital literacy.
  • Perfect for family game nights, classrooms, or on-the-go fun.

How to Play Code Battle

  1. Each player is given cards with HTML tags (e.g., <p>, <h1>, <body>).
  2. Players use their cards to write HTML code structures in 2 minutes.
  3. Points are awarded for correct syntax and positioning of tags (bonus for creativity!).
  4. The player with the highest points wins the round!
